#PyOEIS
PyOEIS is a module for interfacing with the [Online Encyclopedia of Integer Sequences](http://oeis.org)

##Installation
The easiest way to install is with [pip](https://pip.pypa.io/en/latest/).
    
    $ pip install pyoeis
